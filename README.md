# Online Examination & Result Portal

## Developed By
- DILEEP – UI Design (HTML, CSS)
- CHETHAN – JavaScript Logic
- SNUHITH – Result System, GitHub & Documentation

## Project Description
This is a client-side online examination system developed using HTML, CSS, and JavaScript.

## Features
- 10+ MCQ Questions
- Countdown Timer
- Auto Submit
- Score Calculation
- Grade Assignment (A/B/C/Fail)
- Result Page with Percentage

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## How to Run
1. Download project
2. Open index.html in browser
3. Enter name
4. Start Exam
